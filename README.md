# Robot Lang

DSL для управления роботом.

## Установка

1. Скачай `install.bat` и `robot.exe` из [релиза](https://github.com/CeTzY/Robot/releases)
2. Положи оба файла в одну папку (например `C:\RobotLang`)
3. Нажми правой кнопкой на `install.bat` → **Запуск от имени администратора**
4. Всё готово

## Как создать .bot файл

1. Создай текстовый файл (блокнот, VS Code, что угодно)
2. Напиши команды (смотри пример ниже)
3. Сохрани с расширением `.bot` (например `figure.bot`)

## Использование

**Способ 1 — консоль:**
```
robot figure.bot
```

**Способ 2 — двойной клик:**
Просто дважды кликни по `.bot` файлу.

## Пример

Создай файл `example.bot`:

```
color red
width 2
forward 100
right 90
forward 100
right 90
forward 100
right 90
forward 100
```

Запусти:
```
robot example.bot
```

## Команды

| Команда | Описание |
|---------|----------|
| `forward N` | Вперёд на N пикселей |
| `back N` | Назад на N пикселей |
| `right N` | Поворот направо на N градусов |
| `left N` | Поворот налево на N градусов |
| `color имя` | Цвет линии (red, blue, green, yellow...) |
| `width N` | Толщина линии |
| `penup` | Поднять перо (двигаться без рисования) |
| `pendown` | Опустить перо (рисовать снова) |
| `// ...` | Комментарий |

## Настройки в .bot файле

В начале `.bot` файла можно указать настройки:

```
name = "МойРобот"
stopped = 0.3
close = false
```

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| `name` | Имя робота (и лог-файла) | Robot |
| `stopped` | Скорость движения (0 — мгновенно, 1 — медленно) | 1 |
| `close` | Закрыть окно после выполнения (true/false) | true |

## Логи

Все действия записываются в `%APPDATA%\RobotLang\logs\ИмяРобота.log`.
